Из сторонних инструментов были использованы Mirror (из Asset Store) и TextMeshPro.
Кнопки Play и Online переключают приложение на одну и ту же сцену, но с разными условиями.
В онлайн-режиме можно подключиться к другому игроку по IP, если он находится в этой же локальной сети. Для удобства соединения IP адрес текущего устройства (или ПК) выведен на экран.
UI во время подключения - это немного кастомизированный NetworkManagerHUD, поэтому он визуально отличается от остального UI игры.
Начать хостить можно, нажав кнопку Host. После этого клиент может подключиться к IP хоста, введя его в поле для ввода и нажав кнопку Client.
Если клиент не обнаружит хоста по заданному адресу, он будет переключен на сцену меню.
Хост не может нажать кнопку Start, пока нет клиента, только выйти в меню.
Раунд в онлайн-игре начинается, когда и хост, и клиент нажали кнопку Start.
Раунд в онлайн-игре закончится, когда мяч вылетит за границу сверху или снизу. Для продолжения оба игрока снова должны нажать кнопку Start.
Хоста или клиента переключит на сцену с меню, если один из них во время игры отключится или выйдет в меню.
Если игроки с разных устройств нажмут кнопку Host, это не повлечет последствий. Они будут так же ожидать подсоединения клиента.
Соотношение сторон экрана в сцене игры зафиксировано для гарантии одинакового размера поля на всех устройствах.

Проверка работоспособности на IOS не проводилась, поскольку нет подходящего устройства.
